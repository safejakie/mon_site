# Bienvenue sur mon site personnel ! 

Ceci est le dépôt de mon site web personnel, un espace où je présente mon parcours, mes compétences et ce qui me passionne au quotidien.

##  À propos de ce projet
Ce site a été conçu pour être le reflet de mon identité numérique. Il regroupe :
-  **Mon Parcours** 
-  **Mes Passions** 
-  **Mes Centres d'intérêts** 

##  Technologies utilisées
- **HTML5 / CSS3** (pour la structure et le style du site)
- **Bootstrap 5** (pour le design responsive)
- **Font Awesome** (pour les icônes)
- **JavaScript** (pour les interactions)
